IM Client Settings

Here are the settings you need to configure your Pidgin IM client:

Login Options

  • Protocol: XMPP
  • Username: username you specified when creating your account
  • Domain: boykisser.com
  • Resource: any unique string of text that identifies your client

New Account Creation

  • Protocol: XMPP
  • Username: preferably your Active Directory username; however, it can be anything unique.
  • Domain: boykisser.com
  • Resource: any unique string of text
  • Check "create this new account on the server." It will tell the server to create the account with the settings you chose.

SSL Certificate Verification

If you receive a dialog box that asks you to accept the certificate for boykisser.com, select "accept."
